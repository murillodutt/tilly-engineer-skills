# Current Evidence

This directory answers what is currently claimed from retained evidence. It is a small control panel, not a storage area for raw traces.

| File | Role |
|------|------|
| `CLAIMS.md` | Current evidence-backed claims and proof links. |
| `RISKS.md` | Known limits, stale-evidence risks, and migration hazards. |
| `../INDEX.md` | Retention policy for current, reports, and archive layers. |

Agents should start here when they need current evidence interpretation, then follow proof links into `docs/evidence/reports/**` only when the underlying run details are needed.
