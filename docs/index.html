TES installs an assisted context mesh into a target project. Runtime files stay thin. Project truth moves into versioned Markdown under docs/agents/**, with evidence, rollback instructions, and a read-only Cortex MCP server for local recall.

The installer starts with Git status, classifies the project as new, existing, or meshed, preserves project-owned governance, activates project-scoped helpers, and closes with a certification report.
