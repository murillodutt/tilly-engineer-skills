## How to read this prompt

- **Per-unit evidence block** turns each commit into an auditable closure, not a
  claim — the senior-validation spine you asked for.
- **Adversarial self-check** binds the installed protocol back to the five lived
  cases that dictated the design, so the change is validated against the
  experience that produced it, not just against its own prose.
- **Sealed-region rule** is explicit in three places (frozen list, SPEC-050,
  stop conditions) because hand-editing a sha256 region is the one mechanical
  way this change can silently corrupt the bootloader.
- **Honest enforcement** is preserved: the oracle is Stage A string validation;
  the prompt never asks it to detect a violation in a diff it cannot see.
